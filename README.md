# JD Parser API

A REST API that parses raw job descriptions and extracts structured data — job title, skills, experience, location, and work type. Built with Node.js, TypeScript, Express, PostgreSQL, and Prisma.

## Tech Stack
- Node.js + TypeScript
- Express.js
- PostgreSQL (Neon)
- Prisma ORM
- Zod validation

## Getting Started

### Prerequisites
- Node.js v18+
- A PostgreSQL database (or Neon free tier)

### Installation
```bash
git clone https://github.com/YOUR_USERNAME/jd-parser-api
cd jd-parser-api
npm install
```
